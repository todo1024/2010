课程介绍：

互联网刚开始是安全的，但是伴随着黑客的诞生，互联网变得越来越不安全。与此同时，随着Web技术发展越来越成熟，而非Web服务越来越少的暴露在互联网上，现在互联网安全主要指的是Web安全。

为了自身不“裸奔”在大数据里，渐渐开始学习Web安全，在学习Web安全的过程中，发现很大一部分知识点都相对零散，如果没有相对清晰 的脉络作为参考，会给学习带来一些不必要的负担。
